# Replit Notes App

Simple full-stack notes app (Express + SQLite + static frontend).

## Run locally
1. `npm install`
2. `npm start`
3. Open http://localhost:3000

## On Replit
- Create a new Node.js Repl
- Add files as above
- `package.json` start script is `node server.js`
- Replit will run it and expose a URL like: `https://<your-repl>.repl.co`

## Features
- Create / edit / delete notes
- Optional file upload (stored in /uploads)
- SQLite DB (data.db)
